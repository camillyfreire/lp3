# Projeto Gerenciador de Tarefas

Este projeto foi desenvolvido em Python para a disciplina de LP3.

Ele implementa as operações CRUD, seguindo uma arquitetura MVC.

O banco de dados utilizado neste projeto é o **PostgreSQL**.

Para rodar este projeto, é necessário primeiro criar um banco de dados chamado `Projeto_Tarefas` no seu servidor PostgreSQL.
